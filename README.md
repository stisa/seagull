# Seagull

Simple GUI library for Elixir language based on wxErlang.

## Examples

To run the examples, copy **demo** to your **lib** folder, then do:

```
mix compile
iex -S mix
Demo./*moduleName*/.start
```

for example, to start the Button demo, do:

```
mix compile
iex -S mix
Demo.Button.start
```
